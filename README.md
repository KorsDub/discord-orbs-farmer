# Discord Orbs Farmer

**v8.1 — Простое меню** ✨

Теперь с **очень простым и удобным меню**. Одно нажатие — и фарм пошёл.

## Что это такое?

Userscript для автоматического фарма **Discord Orbs** через выполнение квестов (видео + игры).

## 🚀 Быстрая установка

1. Установи **Tampermonkey**
2. Открой Discord Web (discord.com/app)
3. Создай новый скрипт в Tampermonkey
4. Вставь содержимое файла `discord-orbs-farmer.user.js`
5. Сохрани и обнови Discord

## ✨ Что нового в v8.1

- **Простое меню** — минимум элементов, максимум удобства
- Большая кнопка СТАРТ — сразу начинает фарм
- Автоматический выбор лучших квестов (видео в приоритете)
- Компактные настройки в один клик
- Всё остальное (логика, защита от банов, умные паузы) осталось мощным и надёжным
- Полная перезапись с нуля + упрощённый интерфейс

## Как пользоваться (ещё проще)

1. Открой Discord
2. Нажми большую синюю кнопку **СТАРТ ФАРМА**
3. Скрипт сам выберет квесты и начнёт работать
4. Чтобы остановить — нажми **СТОП**
5. Настройки — маленькая кнопка ⚙

## Настройки (простые)

- Уровень (Быстрый / Баланс / Безопасный)
- Автопринятие квестов
- Автоклейм наград
- Без пауз (риск)
- Уведомления
- Скорость видео

## ⚠️ Важно

- Только для личного использования
- Лучше использовать уровень **Баланс**
- Работает только в веб-версии Discord
