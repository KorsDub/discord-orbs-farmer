## Что это такое?

**KDStudio Discord Orbs Farmer** — лёгкий инструмент для Windows, который создаёт фейковые процессы игр.  
Discord определяет игры, глядя на имена запущенных процессов. Этот инструмент просто создаёт процесс с точным именем, которое Discord ожидает увидеть.

Он **не**:
- Внедряет код в Discord
- Изменяет файлы клиента Discord
- Отправляет фейковые запросы к API Discord изнутри клиента
- Требует каких-либо модификаций клиента (Vencord, BetterDiscord и т.д.)

Он только создаёт переименованный процесс в вашей системе.

## Возможности

- Загружает актуальный список определяемых игр из официального API Discord
- Резервный источник (GitHub) на случай недоступности API
- Поиск игр по названию или аббревиатуре
- Ручной режим (можно ввести любое имя .exe)
- Создаёт фейковые процессы в папке `Desktop\Win64`
- Можно запускать несколько игр одновременно
- Красивый цветной интерфейс в терминале
- Легко собирается в один `.exe` файл (после сборки Python не нужен)

## Быстрый старт (из исходников)

```bash
git clone https://github.com/KorsDubStudio/discord-orbs-farmer.git
cd discord-orbs-farmer
pip install -r requirements.txt
python kdstudio_orbs.py
```

---

## Сборка standalone .exe (рекомендуется)

После сборки программа будет работать на любом компьютере с Windows **без установленного Python**.

```bash
pip install -r requirements.txt pyinstaller
pyinstaller --onefile --noconsole --name "KDStudio_Orbs" kdstudio_orbs.py
```

Готовый файл появится здесь:

```
dist/KDStudio_Orbs.exe
```

Его можно копировать куда угодно и запускать.

## Как пользоваться

1. Убедитесь, что **Discord запущен**.
2. Запустите `KDStudio_Orbs.exe` (или `python kdstudio_orbs.py`).
3. Выберите пункт **1** — поиск по базе Discord.
4. Введите название игры (например: `Roblox`, `Fortnite`, `Valorant`).
5. Выберите нужную игру из списка.
6. Подтвердите создание фейкового процесса.
7. Оставьте процесс работать нужное время (обычно 15 минут).
8. Можно запускать несколько игр сразу, просто повторив процесс.

Фейковые исполняемые файлы создаются в:

```
%USERPROFILE%\Desktop\Win64\
```

---

## Ручной режим

Если игра не найдена в базе, выберите пункт **2** и введите точное имя процесса, которое ожидает Discord (например `RobloxPlayerBeta.exe`).
