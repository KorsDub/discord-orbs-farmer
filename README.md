## 🚀 Как использовать

1. Discord Desktop / Vesktop / Equicord
2. `Ctrl+Shift+I` → Console → `allow pasting`
3. Вставь код из `discord-orbs-farm-stealth.js`
4. Настрой в ⚙ и нажми **СТАРТ**
5. ✕ — полностью закрыть

---

## ⚙ Настройки

- Три режима скорости/безопасности
- Авто-принятие квестов
- Приоритет видео
- **Новый:** Убрать паузы между квестами (быстрее, но рискованнее)
